# 入库模块文档目录

本目录用于存放入库模块相关设计与计划文档。

## 文档清单

- `inbound-module-design.md`：入库模块功能设计文档
- `inbound-module-plan.md`：入库模块开发计划文档
- `inbound-module-cards.md`：入库模块执行卡片（含状态表）
- `inbound-module-cr.md`：每张 Card 的独立 CR 审查记录
- `inbound-module-commits.md`：Commit 文件清单与 Message 建议
- `inbound-module-known-issues.md`：本轮所有已知遗留问题、范围取舍与后续优化建议总表
- `inbound-module-demo.md`：课堂/汇报演示流程文档
- `inbound-module-final-acceptance.md`：最终验收记录与交付结论
