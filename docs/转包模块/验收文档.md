## 附录：API测试命令

```bash
# 查询可转包看板
curl -X GET "http://localhost:8080/api/transfer/kanbans?page=1&size=10" \
  -H "Authorization: Bearer {token}"

# 查询转包历史
curl -X GET "http://localhost:8080/api/transfer/history?page=1&size=10" \
  -H "Authorization: Bearer {token}"

# 执行转包
curl -X POST "http://localhost:8080/api/transfer/execute" \
  -H "Authorization: Bearer {token}" \
  -H "Content-Type: application/json" \
  -d '{"sourceKanbanNo":"K-001","transferQty":30}'
```
