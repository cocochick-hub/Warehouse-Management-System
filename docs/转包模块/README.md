## 功能概述

转包功能实现"出库→新建小箱看板→入库"的完整闭环流程：

1. **源看板出库**：创建 `OutboundOrder` + `OutboundOrderDetail`，扣减库存
2. **目标看板入库**：创建新的 `InboundOrder` + `InboundOrderDetail`，增加库存
3. **目标看板标签**：创建新的 `InboundKanbanLabel`，关联新入库单
4. **状态更新**：更新源看板状态为"转包"或"已转包"
5. **历史记录**：记录转包操作历史

## 技术栈

- **后端**：Spring Boot 2.7.18, Spring Data JPA, MySQL
- **前端**：Vue 3, Element Plus, Vite
- **移动端**：UniApp, Vant
